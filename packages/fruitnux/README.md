# @leapnux/fruitnux

External-deliverables CLI for the 6-NUX taxonomy. SCAs, audit packages, regulator-facing PDFs, sign-off packets.

**Status:** v0.4.0-alpha.1 — package skeleton. Planned for **v0.5.0**.

## Where this fits

fruitnux is the **external deliverables layer** of the 6-NUX taxonomy:

```
root → trunk → branch → leaf → FRUIT (fruitnux)  → soil
```

The harvest. What outsiders consume — auditors, regulators, customers, investors. SCAs, pen-test reports, SOC 2 attestation packages, KPI dashboards, sign-off packets.
